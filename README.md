# 안내견학교 데이터 관리 시스템

NVDA 스크린리더 접근성을 최우선으로 고려한 안내견학교 데이터 관리 웹 애플리케이션입니다.

## 주요 기능

### 1. 데이터 작성
안내견, 파트너, 활동 정보를 한 번에 입력할 수 있는 폼을 제공합니다.

**안내견 정보**
- 견명 (필수)
- 생년월일 (필수)
- 성별 (필수)
- 모견 (어미견)
- 부견 (아비견)
- 퍼피티칭 시작일/종료일
- 훈련 시작일/종료일

**파트너 정보**
- 파트너 성명 (필수)
- 파트너 성별 (필수)
- 연락처 (필수)
- 주소 (필수)
- 파트너 생년월일 (필수) - 입력 시 자동으로 나이 계산
- 직업 카테고리 (필수): 학생, 회사원, 교직, 자영업, 주부, 종교인, 기타
- 직업 상세

**활동 정보**
- 활동 시작일 (필수)
- 활동 종료일

### 2. 데이터 조회
등록된 모든 데이터를 테이블 형태로 조회할 수 있습니다.
- 실시간 데이터 업데이트
- 데이터 삭제 기능
- 접근성 높은 테이블 구조

## 접근성 기능

이 애플리케이션은 NVDA 스크린리더 사용자를 위해 다음과 같은 접근성 기능을 제공합니다:

### 1. 폼 접근성
- 모든 입력 필드에 명확한 레이블 연결
- 필수 항목 표시 (별표 * 와 aria-required 속성)
- 에러 메시지를 aria-describedby로 연결
- aria-invalid 속성으로 유효성 오류 표시
- 에러 발생 시 첫 번째 에러 필드로 자동 포커스 이동

### 2. 키보드 내비게이션
- Tab 키로 모든 요소 탐색 가능
- Enter/Space 키로 버튼 실행
- 명확한 포커스 표시 (파란색 링)

### 3. 스크린리더 지원
- 의미론적 HTML 구조 (section, fieldset, legend)
- ARIA 속성 활용 (role, aria-label, aria-live)
- 동적 콘텐츠 변경 시 알림 (aria-live)
- 스킵 링크로 메인 콘텐츠 바로가기

### 4. 테이블 접근성
- 테이블 캡션 제공
- 헤더 셀에 scope 속성 지정
- 명확한 구조로 스크린리더 내비게이션 지원

## 기술 스택

- **React 18** - UI 라이브러리
- **TypeScript** - 타입 안전성
- **Vite** - 빌드 도구
- **Tailwind CSS** - 스타일링
- **LocalStorage** - 데이터 저장

## 설치 및 실행

### 요구사항
- Node.js 18 이상
- npm 또는 yarn

### 설치
```bash
cd guidedog-system
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

개발 서버가 실행되면 브라우저에서 `http://localhost:5173` 으로 접속하세요.

### 빌드
```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

### 프리뷰
```bash
npm run preview
```

## 데이터 저장

현재 버전은 브라우저의 **LocalStorage**를 사용하여 데이터를 저장합니다.

### 주의사항
- 데이터는 브라우저에만 저장됩니다 (서버 없음)
- 브라우저 캐시를 지우면 데이터가 삭제됩니다
- 동일한 컴퓨터의 동일한 브라우저에서만 데이터를 볼 수 있습니다

### 향후 개선 방향
- 백엔드 API 연동으로 다중 사용자 지원
- 데이터베이스 연동으로 영구 저장
- 데이터 내보내기/가져오기 기능

## NVDA 스크린리더 사용 팁

### 폼 작성
1. Tab 키로 다음 입력 필드로 이동
2. Shift + Tab 키로 이전 입력 필드로 이동
3. 필수 항목은 "별표 필수"로 읽힙니다
4. 에러가 있으면 에러 메시지가 자동으로 읽힙니다

### 테이블 탐색
- **Ctrl + Alt + 화살표 키**: 테이블 셀 이동 (NVDA)
- **Ctrl + Alt + Shift + 위/아래**: 헤더 읽기
- **화살표 키**: 셀 단위 이동
- **Enter 또는 Space**: 삭제 버튼 실행
